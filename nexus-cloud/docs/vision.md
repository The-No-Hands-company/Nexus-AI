# Vision

Nexus Cloud is the cloud substrate for The No Hands Company.

## Goal

Build a self-hosted, federated cloud that can be run by one person, a team, or a community without depending on a hyperscaler.

## First principles

- Sovereign by default
- Federated when needed
- Portable workloads
- Simple operator control
- No vendor lock-in

## Initial building blocks

1. Identity and trust
2. Node registration
3. Scheduling
4. Storage
5. Networking
6. Observability
