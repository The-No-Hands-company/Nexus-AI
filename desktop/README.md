# Desktop App

This directory contains the skeleton for a desktop application that connects to the Nexus AI backend.

## Planned Features
- Cross-platform desktop app (Electron, Tauri, or native)
- Authentication and user management
- Chat interface with AI agents
- Voice input/output capabilities
- Image generation and viewing
- Collaboration features
- Access to proactive agents
- System tray integration
- Native notifications
