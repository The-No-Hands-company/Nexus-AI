# Nexus-AI Sovereign Model Plan

This document contains the long-horizon model sovereignty plan, separated from the execution roadmap.

## Goal

Build and operate Nexus-owned model capability over time while preserving self-hosted and privacy-first principles.

## Stage 1: Fine-Tuned Foundation

- Fine-tune strong open-weight bases for Nexus-specific tasks
- Add reproducible eval suite for coding/reasoning/retrieval tasks
- Integrate trained adapters into default router paths safely

## Stage 2: Continuous Improvement Loop

- Preference learning from explicit user feedback signals
- Regular retraining cadence with benchmark-based promotion gates
- Distillation and specialization adapters by task profile

## Stage 3: Purpose-Built Architecture

- Scale data curation and model architecture work
- Build robust training/inference infrastructure with reliability controls
- Publish model card, safety notes, and evaluation results

## Stage 4: Federated and Planetary Scale (Vision)

- Explore privacy-preserving distributed learning approaches
- Consider optional user-contributed compute under strict safeguards
- Preserve data minimization and sovereignty principles

## Governance Constraints

- No user data collection without explicit consent controls
- No roadmap milestone promotion without measurable eval gains
- Keep product UX independent from sovereignty research cadence
